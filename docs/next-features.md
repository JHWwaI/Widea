# Widea — 다음 기능 로드맵

오늘 자고 일어나면 보실 우선순위 정리. 카테고리별로 (1) 진짜 필요한 것 (2) 눈 휘둥그레지는 차별화 기능 분리.

## 🟢 진짜 필요한 것 (사용자가 떠나지 않게 막는 기능)

### A. 아이디어 매칭 신뢰도

| # | 기능 | 왜 필요 | 구현 난이도 |
|---|---|---|---|
| A1 | **Llama → Gemini 1.5 Pro / Claude 3.5 Sonnet 교체** | Llama-3.3-70B는 instruction following이 약해서 매번 컨설턴트 톤·반복 패턴이 새어나옴. 모델 한 줄 바꾸면 30~50% 품질 점프 | 30분 |
| A2 | **archive_link 클릭 시 새 탭 + Wayback 스냅샷 검증** | 현재 일부 archive_link가 `web/2018/` 같이 정확한 타임스탬프 없는 형식. 클릭 시 안 열리면 신뢰 깎임. 백그라운드 검증 + 캐시 | 2시간 |
| A3 | **vector 검색 결과에 "왜 이 회사가 매칭됐나" 설명** | 현재 매칭 점수만 보여줌. "이 산업/모델/타겟이 일치해서" 한 줄 설명이 LLM 매칭 신뢰의 근거 | 1시간 |

### B. 결과 → 실행 전환

| # | 기능 | 왜 필요 | 구현 난이도 |
|---|---|---|---|
| B1 | **idea-match 결과 PDF 다운로드** | 사용자가 친구·투자자에게 보여줘야 함. 현재 화면 캡처가 답인데 너무 조잡 | 3시간 |
| B2 | **공유 가능한 read-only 링크** | `/ideas/[id]/public?token=xxx`로 비로그인 친구가 볼 수 있음. 바이럴 트리거 | 4시간 |
| B3 | **artifacts 카드를 실제 워크스페이스에 자동 적재** | 현재는 복사 + 별도 입력. "이대로 프로젝트 만들기" 한 번에 → /projects/new 자동 프리필 | 2시간 |

### C. 워크스페이스 (외주·팀)

| # | 기능 | 왜 필요 | 구현 난이도 |
|---|---|---|---|

## 🔴 눈 휘둥그레질 차별화 기능 (다른 서비스가 없는 것)

### D. AI가 진짜 일하게 만드는 기능

| # | 기능 | 임팩트 | 구현 난이도 |
|---|---|---|---|
| D1 | **"오늘 할 일" AI 비서** — idea 생성 후 매일 1개씩 "오늘 이걸 하세요" 푸시 (예: "오늘은 잠재 고객 5명에게 LinkedIn DM 보내기 — 메시지 초안 자동 생성") | 사용자가 매일 들어오게 하는 retention 도구 | 2일 |
| D2 | **벤치마크 회사의 트위터/뉴스 실시간 모니터링** — Stripe·Linear의 새 발표가 있으면 "당신 idea와 관련 있는 변화" 알림 | "이 서비스만 알려준다" 차별화 | 3일 |
| D3 | **AI 가설 검증 인터뷰 봇** — 잠재 고객 5명에게 자동 음성 인터뷰 전화 → 결과 요약. 한국 ARS API + Whisper + GPT 조합 | 다른 서비스에 없음. 실제 PMF 검증 자동화 | 1주 |
| D4 | **벤치마크 기반 자동 랜딩페이지 생성** — idea 입력 → 실제 작동하는 미니 랜딩 + 결제 모듈 (Stripe Checkout) 5분 안에 배포 | "아이디어 → 검증" 시간을 일주일에서 30분으로 | 1주 |

### E. 커뮤니티/네트워크 효과

| # | 기능 | 임팩트 | 구현 난이도 |
|---|---|---|---|
| E1 | **AI 팀원 매칭** — community/팀원모집 글 작성 시 AI가 자동으로 보유 사용자 풀에서 적합한 사람 5명 추천 + DM 발송 | 단순 게시판이 아니라 "매칭"되는 데이터베이스 | 2일 |
| E2 | **"아이디어 클로너" 알람** — 누군가 비슷한 idea를 만들면 "당신과 80% 일치하는 idea가 어제 만들어졌습니다 → 협업/경쟁 알림" | 서비스 안에서의 우연한 만남 유도 | 2일 |
| E3 | **AC(액셀러레이터)·VC가 직접 보는 "유망 idea 피드"** | 액셀이 사용자가 되면 idea 작성자에게 즉시 컨택 → 매출 모델 추가 | 5일 |
| E4 | **Founder Match (Tinder for cofounder)** — 사용자 프로필·idea 호환성 점수 매겨 1:1 매칭 | 팀빌딩이 핵심 페인 | 3일 |

### F. 데이터·차트가 살아있게

| # | 기능 | 임팩트 | 구현 난이도 |
|---|---|---|---|
| F1 | **벤치마크 회사 ARR 성장 곡선 (Sankey/실시간 업데이트)** | 텍스트 대신 곡선 — "Stripe 2010 ARR $0 → 2024 ARR $14B" 시각화 | 1일 |
| F2 | **사용자 idea의 "시장 지도" 자동 그리기** — 100개 시드 위에 사용자 idea 위치를 점으로, 거리 = 유사도 | 시각화 자체가 인스타용 | 2일 |
| F3 | **CAC/LTV 대시보드 시뮬레이터** — 사용자가 슬라이더 조정하면 실시간으로 break-even 시점 그래프 변동 | 단위경제를 직관적으로 학습 | 2일 |

### G. "이건 진짜 처음 본다" 카테고리

| # | 기능 | 임팩트 | 구현 난이도 |
|---|---|---|---|
| G1 | **idea를 음성으로 입력 (3분 녹음 → AI가 정제 → 사업계획서)** | "타이핑 귀찮아"의 페인 정공 | 3일 |
| G2 | **벤치마크 회사 창업자가 "당신 idea를 보면 뭐라고 할까?" AI 시뮬레이션** — Patrick Collison(Stripe) AI persona가 사용자 idea에 코멘트 | 기획자에 강하게 어필. 디지털 멘토 컨셉 | 4일 |
| G3 | **실시간 idea 라이브 — 다른 사용자가 지금 어떤 idea를 만들고 있는지 (익명) 화면 한쪽에 흐르게** | "나 혼자 아니구나" 감정 + 경쟁/영감 | 2일 |
| G4 | **idea 가치 경매 (베타 사용자 사이)** — 만든 idea를 $X에 다른 창업자에게 판매. 검증 후 가치 책정 | 저장된 idea가 자산이 되는 새 카테고리 | 1주 |

## 🎯 내가 추천하는 다음 1주 우선순위

**Day 1 (월): A1 + A3 + C1**
- 모델 교체 (Llama → Gemini Pro 또는 Claude)
- 매칭 이유 표시
- community 프리필 검증
→ **이거 하나로 사용자가 느끼는 품질이 가장 크게 점프**

**Day 2~3 (화·수): D1 + B2**
- "오늘 할 일" AI 비서 — retention 핵심
- 공유 링크 — viral 핵심

**Day 4~5 (목·금): F2**
- 시장 지도 시각화 — 인스타에 자랑할 수 있는 자산

이 5일치만 하면:
1. 품질 (다른 서비스보다 명확히 좋다)
2. 매일 들어옴 (retention)
3. 친구한테 보여줌 (viral)

세 박자가 맞아서 **"실제로 쓰는 사람"이 생기기 시작**합니다. 그 다음에 D2·D3·D4 같은 큰 차별화로 갑니다.

## 🔧 지금 켜져 있는 백엔드 기능 (재확인)

- ✅ `/api/idea-match` — 3개 idea 생성 + 환각 post-filter + 컨설턴트 톤 차단
- ✅ `/api/ideas/:id/deep-report` (8 크레딧) — 5단계 심층 분석
- ✅ `/api/ideas/:id/artifacts` (5 크레딧) — 모집공고·커피챗·투자체크리스트 3종
- ✅ `/api/ideas/:id/unlock` (5 크레딧) — 잠긴 idea 공개
- ✅ Pinecone 110개 벤치마크 임베딩 (3072차원, gemini-embedding-001)
- ✅ Postgres GlobalCaseMeta + DeepAnalysis (rawSeedData 포함)
